*********JOGO DE FORCA*********

Este jogo permite que o usuário chute as letras 
e acerte a palavra tendo até 5 chances de errar.
Caso ele acerte ele irá ganhar, ou caso nas 5 
tentativas não consiga descobrir a palavra secreta,
ele irá perder.

Ao final é possível se adicionar uma sugestão de nova
palavra para nosso banco de dados.
